This data set contains 80X80 pixel JPEG images of Cellular Automata (CA). It includes 5 trials of length 100 - 200 (iterations) for each of the 4 classes of CA defined by Wolfram, extended to 2D CA.

5 trials for each of 4 classes

class 1:

    initial conditions: 80 X 80 grid, random with 60% bias

    rule: if less than 5 neighboring cells are on, turn current cell off, otherwise turn it on.

 

class 2:

    initial conditions: 80 X 80 grid, random with 30% bias

    rule: if less than 4 neighboring cells are on, turn current cell off, otherwise turn it on.

 

class 3:

    initial conditions: 40 X 40 grid, random with 30% bias

    rule: standard game of life rules:

 

class 4:

    initial conditions: 47 X 47 grid, random with 50% bias

    rule: if number of neighboring cells that are on is even, turn the current cell on, if odd, turn current cell off
